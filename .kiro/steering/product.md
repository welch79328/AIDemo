# Product Steering

## Purpose
業務行動成效評估系統旨在幫助 JGB Smart Property（金箍棒智慧物管）的業務團隊系統化管理客戶拜訪流程，智能評估客戶成熟度，並追蹤導入成效。

**核心用戶**: 業務人員、業務主管、系統管理員

**解決問題**:
- 傳統客戶拜訪資料散亂，難以系統化整理與分析
- 無法量化評估客戶優先級與成熟度
- 缺乏業務行動成效追蹤機制

## Value Proposition

### 智能客戶評估
基於「客戶健檢表 30 問」架構，系統化記錄一訪/二訪問卷資料，自動計算客戶成熟度評分（0-100 分），並智能標記 AA 客戶（高價值客戶）。

### 業務成效量化
提供個人與團隊業務指標儀表板，追蹤拜訪次數、轉換率、AA 客戶獲取等關鍵指標，協助管理層做出數據驅動決策。

### AI 輔助分析
整合 OpenAI API，提供對話記錄分析、客戶資訊自動擷取、AA 客戶評估等 AI 功能，大幅減少手動整理時間。

## Core Capabilities

### 1. 客戶生命週期管理
- **客戶分類**: 個人戶、準備成立公司、剛成立公司、規模擴張等階段
- **狀態追蹤**: 已接觸 → 一訪排程 → 一訪完成 → 二訪排程 → 二訪完成 → 洽談中 → 已簽約
- **AA 客戶標記**: 基於 4 個關鍵條件自動識別高價值客戶

### 2. 拜訪記錄與問卷管理
- **一訪問卷**: A 類（公司基本資料）、B 類部分（物件狀態）
- **二訪問卷**: B 類完整（發展方向）、C 類（經營狀態）、D 類（數位化程度）
- **問卷分類**: 必問、建議、可選（Nice to have）三級優先級

### 3. AI 分析功能
- **對話分析**: 分析業務與客戶對話內容，提取關鍵資訊
- **客戶資訊擷取**: 從文字稿自動擷取公司名稱、聯絡人、電話等
- **AA 客戶評估**: 基於業務 30 問自動評估客戶等級

### 4. 簽約與導入追蹤
- **合約類型**: 包租、代管、代租、混合
- **導入 KPI**: 追蹤物件上傳率、合約建立率、帳務發送、金流整合、自動通知、SOP 建立等 6 大指標
- **導入成功判定**: 基於預設門檻值（如物件上傳率 ≥ 50%）

### 5. 業務績效儀表板
- **個人指標**: 拜訪次數、轉換率、AA 客戶獲取
- **團隊指標**: 整體漏斗轉換、平均成交週期
- **時間維度**: 日、週、月、季、年度統計

## Domain Concepts

### 客戶健檢表 30 問
源自「5.00.8.3_導入顧問服務_業務30問_客戶健檢表.xlsx」，分為四大類別：
- **A 類**: 公司基本資料（一訪）
- **B 類**: 物件狀態與發展（一訪部分 + 二訪完整）
- **C 類**: 經營狀態（二訪）
- **D 類**: 數位化程度（二訪）

### AA 客戶判定條件
高價值客戶識別基於四個關鍵條件：
1. 物件數量 ≥ 門檻值
2. 數位化程度評分 ≥ 門檻值
3. 成長意願強烈
4. 決策力強（可快速做決定）

### 導入成功 KPI
基於「導入成功KPIs」工作表定義的六大指標：
1. 物件管理（上傳率 ≥ 50%）
2. 合約管理（建立率 ≥ 50%）
3. 帳務管理（定期發送）
4. 金流整合
5. 自動通知
6. SOP 操作

### 成熟度評分
0-100 分量化評估客戶在數位化、經營規模、成長意願等維度的綜合表現，協助業務優先排序。

## User Experience

### 主要工作流程

**新客戶開發**:
1. 建立客戶資料（匯入或手動建立）
2. 排程一訪 → 完成一訪問卷（A 類 + B 類部分）
3. 系統初步評分 → 判斷是否進入二訪
4. 排程二訪 → 完成二訪問卷（B 類完整 + C 類 + D 類）
5. 系統計算成熟度評分 → 自動標記 AA 客戶
6. 進入洽談 → 簽約 → 導入追蹤

**AI 輔助分析**:
1. 上傳業務對話錄音或文字稿
2. AI 自動分析對話內容，匹配業務 30 問
3. 擷取客戶關鍵資訊
4. 評估客戶等級，生成健檢報告

**業務成效檢視**:
1. 登入系統 → 查看個人儀表板
2. 檢視本週/本月拜訪次數、轉換率
3. 主管查看團隊整體漏斗、AA 客戶獲取趨勢
